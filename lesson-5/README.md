## 🚀 Команди для запуску проєкту

Виконуйте ці команди по черзі у терміналі для повного розгортання інфраструктури:

```bash
# 1. Вимикаємо віддалений бекенд на перший запуск
mv backend.tf backend.tf.bak

# 2. Налаштовуємо доступ до AWS акаунту
aws configure

# 3. Ініціалізуємо Terraform локально
terraform init

# 4. Розгортаємо базові ресурси (бакет, мережу, ECR)
terraform apply

# 5. Повертаємо конфігурацію бекенду назад
mv backend.tf.bak backend.tf

# 6. Мігруємо локальний стейт-файл у створений S3-бакет на AWS
terraform init -migrate-state
